Bienvenue sur le prototype d'un chat interne dédié à l'agence Médiane Grenoble. Réalisé dans le cadre d'une période inter-contrat afin de tester et de s'autoformer à trois technologies : pico.css, brython et socketio.
Respectivement, la première est une bibliothèque CSS très légère permettant de rapidement faire de l'interface utilisateur. La seconde quant à elle, est une librairie Python-Javascript qui permet de faire du Python directement dans le navigateur client-side.
La dernière, permet d'utiliser les sockets de manière "haut-niveau" avec un fonctionnement event-based.
Brython transpile le Python en Javascript.
